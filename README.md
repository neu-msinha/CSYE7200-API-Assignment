# Spotify Playlist Analyzer

This project fetches data from the Spotify Web API and performs two tasks:

1. Find the Top 10 Longest Songs in a given playlist.
2. Fetch all artists for those 10 songs and display them ordered by number of followers.

The playlist used in this assignment is:

Playlist ID: `5Rrf7mqN8uus2AaQQQNdc1`

---

## Technologies Used
- Scala 3
- sbt
- requests-scala (HTTP client)
- ujson (JSON parsing)
- Spotify Web API
- Functional programming style
